An agent-based network model of injection drug use and HCV

Forrest W. Crawford

Background

Hepatitis C virus

Model

Consider a closed population of \(N\) injection drug users at risk of Hepatitis C Virus (HCV). When an infectious (HCV+) individual shares a needle with a susceptible (HCV-) individual, the virus may be transmitted.

Suppose individuals inject at a constant rate \(\lambda\). When an individual injects, they either inject alone with probability \(1-p\), or inject with a randomly chosen network neighbor with probability \(p\). When an infected individual shares a needle with an uninfected individual, transmission happens with probability \(q\). Assume that HCV infection renders an individual instantly infectious.

Network model

We will represent the needle sharing network as a symmetric contact matrix \(\mathbf{A}\) whose rows and columns correspond to individuals. Assume that links in this matrix are drawn independently from the Bernoulli\((r)\) distribution:

\[ A_{ij} \sim \text{Bernoulli}(r) \]

This is known as the homogeneous, Bernoulli, or Erdos-Renyi, network model.

Goal

We wish to construct a realistic model of injection drug use, needle sharing, and HCV in a small population of injection drug users. We will assume there is one initial HCV infection in the population, and we will assess the spread of infection up to a maximum time horizon. We will use this model to evaluate the possible effects of hypothetical policy/education/healthcare interventions in the population.

Because the population is connected we expect the effects of interventions to be nonlinear and potentially complex.

To help you get started, below is code for an SI process on a network, starting with a single infected individual.

Network SI model

# the population size 
N = 50

# the needle sharing connection probability
# so that average degree is 5
r = 5/(N-1)

# the transmission rate beta 
bet = 0.5 

# max time
tmax = 2

# set the infection indicator 
y = rep(0,N)

# we will set person 1 to be infected at baseline
y[1] = 1 

# set the infection time 
ts = rep(NA,N)
ts[1] = 0

# set up a symmetric zero-diagonal contact matrix 
A = matrix(0, nrow=N, ncol=N)
for(i in 1:(N-1)) {
  for(j in (i+1):N) {
    A[i,j] = rbinom(1,1,r)
    A[j,i] = A[i,j]
  }
}
A[diag(A)] = 0

Now run the simulation. Make sure you understand how it works before changing anything!

tcur = 0

while(tcur < tmax && any(y==0)) {

  # the number of edges connected to each susceptible unit
  # that are attached to an infected unit
  susc_edges = (1-y) * (A %*% y)

  # the waiting time to the next infection 
  w = rexp(1, bet*sum(susc_edges) )

  # if this would put us over the time horizon, stop
  if(tcur + w > tmax) break

  # choose the id of the next infected unit 
  idx = sample(1:N, 1, prob=susc_edges)

  # mark this unit infected 
  y[idx] = 1

  # record the infection time of the newly infected unit 
  ts[idx] = tcur + w

  # update the current time 
  tcur = tcur + w
}

ts[y==0] = Inf

Now lets look at the infection times.

plot(sort(ts), 1:N, bty="n", xlab="Infection time")

Lets plot the network and see how the infection spreads. This is useful for visual intuition, but not necessary to answer any of the questions below.

plot_network_infections = function(t,lay) {
  set.seed(1)
  g = network(A, vertex.attr=list(y=ifelse(ts<=t,1,0)), matrix.type="adjacency", directed=FALSE)
  plot(g, 
       vertex.col=ifelse(get.vertex.attribute(g,"y"),"red","white"), 
       vertex.cex=2, 
       main=paste("t =", round(t,digits=2)))
}

tseq = seq(0,max(ts[ts<Inf]),length.out=9)
par(mfrow=c(3,3), mar=c(1,0,1,0))

for(t in tseq) {
  plot_network_infections(t,lay)
}

Questions

You should create a function that runs the simulation once, with specified parameters. Then, for questions that ask about the average number of HCV infections, you will run this function many times to approximate the average.

  1. What is the average number of infections that occur in the population?
  2. You wish to evaluate the effect of introducing clean syringes into the population. Suppose at each needle sharing event, individuals instead use two clean needles with some probability. How large does this probability need to be in order to reduce the number of HCV infections by half?
  3. Suppose that through a health education campaign, you can reduce the average number of needle sharing contacts of each individual by half, corresponding to \(r/2\). How does this change the average number of infections in the population?
